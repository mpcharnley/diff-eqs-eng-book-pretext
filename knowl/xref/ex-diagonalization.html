We say that a matrix \(A\) is if there exist matrices \(D\) and \(P\) so that \(PDP^{-1} = A\text{.}\) This really means that \(A\) can be represented by a diagonal matrix in a different basis (as opposed to the standard basis). One way this can be done is with eigenvalues.

Find the eigenvalues and corresponding eigenvectors of this matrix. Compute \(E^{-1}\text{.}\) Work out the products \(EDE^{-1}\) and \(E^{-1}AE\text{.}\) What do you notice here?
